# MDate

```
npm i mini-date-format --save

import MDate from 'mini-date-format'

MDate(M,F);

MDate('2015-11-01 15:25:58','YYYY-MM-DD')//  2015-11-01

```


```
@param
M-->'2016-12-01 14:20:01'
F-->'YYYY-MM-DD mm:ss:qq S'
'Y' ----- 'Year'
'M' ----- 'Month'
'D' ----- 'Day'
'h' ----- 'hour'
'm' ----- 'minute'
's' ----- 'second'
'q' ----- 'quarter'
'S' ----- 'millisecond'
```

# MDate

```
npm i mini-date-format --save

import MDate from 'mini-date-format'

MDate(F,M);

MDate('YYYY-MM-DD','2015-11-01 15:25:58')//  2015-11-01

```


```
@param

F-->'YYYY-MM-DD mm:ss:qq S'
'Y' ----- 'Year'
'M' ----- 'Month'
'D' ----- 'Day'
'h' ----- 'hour'
'm' ----- 'minute'
's' ----- 'second'
'q' ----- 'quarter'
'S' ----- 'millisecond'


M--> object Date  or  string Date , default new Date();

```
